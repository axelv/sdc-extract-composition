Composition Template Preview - Alternatives

5 alternative presentations addressing section clarity, WYSIWYG fidelity, and annotation placement.

1. Margin Annotations

PER ITEM and ALS labels move to a left gutter. The document area stays clean and WYSIWYG. Context expressions appear as small annotations in the margin.

Layout

2. Card Containers

Each section is a distinct card with clear boundaries. Badges live in the card header bar only. ALS conditions are indented blocks with colored border.

Layout

3. Color-Coded Bands

Alternating background colors define section boundaries. Small icons replace text labels. A legend explains the visual language.

Layout

4. Split View

Left panel is the pure WYSIWYG document. Right panel shows all technical annotations aligned to their corresponding sections.

Layout

5. Toggle Overlay

Clean WYSIWYG by default. Checkbox toggles reveal/hide section metadata and conditional annotations independently.

Interactive

Issues Addressed

  • Section boundaries unclear — Cards (2), color bands (3), and margin gutters (1) create visual containment
  • PER ITEM / ALS too intrusive — Moved to margins (1), card headers (2), icons (3), separate panel (4), or toggleable (5)
  • Context expression ↔ section link — Gutter alignment (1), header badges (2), context bar (3), side panel (4)
  • WYSIWYG fidelity — Split view (4) and toggle (5) offer the cleanest document view; cards (2) add minimal chrome
